            Customer Decision System
-------------------------------------------------------
1) Install some Python IDE. We recommend PyCharm
2) On windows only run init.bat
3)A virtual enviroment is created and then you can run code with the IDE program.
4)Running process:
    a)open %USERPROFILE%\CustomerSystem\CustomerDecisionSystem with IDE
    b)Configure the interpreter of project to be %USERPROFILE%\CustomerSystem\Scripts\python.exe
    c)Run Program
5)We present the Import and Output files, as long the Excel files and SVGs images.
